Idempotent & Modular Express Routing Repair

An advanced trio of orchestrating tools that inspect, parse, and repair Express.js routing files dynamically with complete duplication safeguard and strict architectural line formatting.

How the Suite Fits Together
Three specialized components acting in alignment.
Imports

Route Imports Fixer

Manages safe insertion of routing imports. Scans files to verify if target declarations exist prior to running mutations. Keeps top-level scope clean and conflict-free.

Consumption

Route Consumption Fixer

Applies endpoint utilization setups (`router.use` or `router.post`). Enforces strict line alignments (zero spacing between routers, correct padding around block bounds).

Orchestration

Orchestrator CLI

Takes high-level declarations and pipelines them to trigger both the Imports and Consumption Fixers sequentially. Produces fully aligned, runnable endpoints instantly.
